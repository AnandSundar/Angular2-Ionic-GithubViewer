<h1>Github Viewer</h1>

<p>This app lets the user to search for GitHub users and their user information using GitHub API</p>

<h2>Technologies Used: </h2>
<ul>
<li>Ionic Framework</li>
<li>GitHub API</li>
<li>Cordova</li>
<li>Angular 2</li>
<li>HTML5/li>
<li>CSS</li>
<li>Node.js</li>
<li>Cordova Plugin: inappwebview</li>
</ul>
